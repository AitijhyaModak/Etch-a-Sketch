
# Etch-a-Sketch

A website that allows you to select grid size and draw sketches.


## Built Using

- HTML

- CSS

- Javascript


## Run this project

Create a project directory
```bash
  mkdir newProject
```

Clone the project

```bash
  git clone git@github.com:AitijhyaModak/Etch-a-Sketch.git
```

Go to the project directory

```bash
  cd newProject
```

Run the index.html file.
